# Basic Calculator

A simple yet powerful command-line **Basic Calculator** written in Python.  
This calculator allows users to perform arithmetic operations—addition, subtraction, multiplication, and division—on any number of inputs provided at once.

---

## Features

- **Multiple Numbers:** Perform calculations on more than two numbers at a time.
- **Operations Supported:** Add, subtract, multiply, or divide.
- **User-friendly:** Interactive terminal prompts make it easy to use.
- **Error Handling:** Gracefully handles invalid input and division by zero.
- **Continuous Operation:** Run multiple calculations in a single session.

---

## How It Works

1. **Select Operation:** Choose addition, subtraction, multiplication, or division.
2. **Enter Numbers:** Input as many numbers as you need, separated by spaces.
3. **Get Result:** The calculator performs the operation and displays the answer.
4. **Repeat or Exit:** Continue with new calculations or exit the program.

---

## Usage

### Prerequisites

- Python 3.x installed on your computer

### Running the Calculator

1. Save the code as `calculator.py` in your desired folder.
2. Open a terminal (Command Prompt, PowerShell, or VS Code Terminal) in that folder.
3. Run the calculator with:
    ```
    python calculator.py
    ```
4. Follow the prompts displayed in the terminal.
